# Atelier Journal

Atelier Journal is a premium editorial blog built with React, Vite, Tailwind CSS, and Framer Motion.

## Stack

- React 19
- Vite 7
- Tailwind CSS 3
- Framer Motion
- React Router DOM

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`

## Notes

The UI was designed as a luxury digital magazine experience with custom motion, editorial typography, and rich mock content.
